# App

## Features

- the user should be able to create, delete and edit todos.
- todos should be grouped in projects.
- when the app is first oppened there should be a default project, where all new todos goes.
- the user should be able to create, delete and edit projects.

## Todo Properties 

- id
- title
- description
- dueDate
- priority
- status

## Project Properties

- id
- name 
- todos
